- Hiển thị webcam góc phải dưới (overlay)
- Gõ code form đăng ký (form trong `register.html`) từng dòng và giải thích các attribute
- Demo: thử 3 lần submit thất bại (validation) và 1 lần submit thành công

- Kết thúc: tóm tắt
